### Install the libraries

```
$ sudo ./INSTALL
```
Eg. Use x11 demo

### Yoloface demo

```
$ ./detect_demo_x11 -m 0 -p 1080p.bmp
```

### YoloV2 demo

```
$ ./detect_demo_x11 -m 1 -p 1080p.bmp
```

### YoloV3 demo

```
$ ./detect_demo_x11 -m 2 -p 1080p.bmp
```

### YoloTiny demo

```
$ ./detect_demo_x11 -m 3 -p 1080p.bmp
```

### YoloV4 demo

```
$ ./detect_demo_x11 -m 4 -p 1080p.bmp
```
### Uninstall the libraries

```
$ sudo ./UNINSTALL
```
